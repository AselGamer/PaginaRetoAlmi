Programacíon

Pagina web

Para crear la pagina web hemos decidido utilizar la IDE de Visual Code y la plataforma de github que hosteara y sorteara las versiones automáticamente. No solo eso pero estamos diseñando la pagina web con flexibilidad y adaptabilidad en mente para que usuarios de todas las resoluciones puedan acceder sin problemas a ella misma. No contentos con eso hemos subido la página a un host web, al mismo tiempo que a un servidor propio.

Github

Para nuestro control de versiones hemos decidido usar github el cual no solo es el mas fácil de utilizar sino que también nos hace el control de versiones y las pull request automáticamente, lo cual nos viene muy bien y hace que podamos hacer backups, ramas diferentes y llevarnos el trabajo a casa sin compromisos.
