Web Server

Operating system and Apache2

When it came to setting up the web server we were presented with several options and after looking for the most suitable one we decided to install Ubuntu server 20.04. On our server we have set up a raid 1 configuration to increase the space not only for the website, but also for the file host. (nextcloud), which will be used to make backups and send important files to other team members.

Firewall

For the firewall (and captive portal) of our network we have used untangle, a free service that anyone with email can use. Our team configured the untangle to create a fast and secure subnet. It also has a captive portal and a VPN server, for which we created a series of clients and downloaded their respective files for use with the mobile and desktop applications, and to add an extra layer of security we also applied mac-blocking.
