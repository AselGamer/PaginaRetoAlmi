Web orrialdea

Webgunea sortzeko IDEa erabiltzea erabaki dugu Visual Code eta plataforma github bertsioak automatikoki ostatatzeko eta marrazteko.Ez hori bakarrik, webgunea malgutasuna eta moldagarritasuna kontuan hartuta diseinatzen ari gara, ebazpen guztietako erabiltzaileek sarbidea izan dezaten. Ez pozik horrekin orrialdea kargatu dugu host web, zerbitzari propio baten aldi berean.

Github

Gure bertsioaren kontrolerako, github erabiltzea erabaki dugu, errazena erabiltzeaz gain, bertsioen kontrola eta pull eskaerak automatikoki egiten ditu, oso ondo etortzen zaigu segurtasun kopiak egiteko, sukurtsal desberdinak egiteko eta gure lana konpromisorik gabe etxera eramatea ahalbidetzen digu.
