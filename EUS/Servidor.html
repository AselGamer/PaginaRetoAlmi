Serbitzari Web

Sistema eragilea eta Apache2

Web zerbitzaria sortzerakoan hainbat aukera aurkeztu zitzaizkigun eta egokiena bilatu ondoren instalatzea erabaki genuen Ubuntu server 20.04. Gure zerbitzarian raid 1 konfigurazio bat egin dugu web orriaren espazioa eta fitxategiaren ostalaria ere anpliatzeko (nextcloud), segurtasun kopiak egiteko eta fitxategi garrantzitsuak beste taldeko kideei bidaltzeko erabiliko dena.

Firewall

Gure sareko suebakia (eta gatibu ataria) erabili dugu untangle, posta elektronikoa duen edonork erabil dezakeen doako zerbitzua. Gure taldeak untangle konfiguratu zuen azpisare seguru eta azkarra sortzeko. Atari gatibu bat eta VPN server bat ere baditu. Horretarako, bezero batzuk sortzen ditugu eta haien artxiboak deskargatzen ditugu, aplikazio mugikorrarekin eta mahaigainekoarekin erabiltzeko. Segurtasuneko geruza gehigarri bat gehitzeko, Macs blokeo bat ere aplikatu dugu.
